# Loops

	<cfloop delimiters="&" index="letter" list="a&b&c">
		<p>#letter#</p>
	</cfloop>


## Attributes

| Name       | Default | Required |
| ---------- | ------- | -------- |
| delimiters | ,       | no       |
| index      |         | no       |
| list       |         | no       |
